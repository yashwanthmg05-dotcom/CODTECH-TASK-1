OVERVIEW OF MUSHROOM CLASSIFICATION:

This project applies machine learning techniques to classify mushrooms as edible or poisonous using the UCI Mushroom dataset. The dataset contains categorical 
attributes such as cap shape, color, and odor, which are preprocessed and fed into models like Random Forest and Naïve Bayes. The goal is to demonstrate how 
supervised learning can be used for classification tasks, achieving high accuracy and providing insights into feature importance.
